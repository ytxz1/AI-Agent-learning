## README 开头建议

README 的前 10 行很重要。

建议包含：

- 项目一句话介绍；
- 技术栈；
- 当前完成的核心功能；
- 快速运行命令；
- 效果截图。

## 截图建议

至少准备：

- FastAPI `/docs` 截图；
- Streamlit 聊天页面截图；
- Docker 启动成功截图；
- 项目目录结构截图。
