# 简历项目经历模板

## 项目名称

AI Agent 实习学习项目

## 简历写法

**AI Agent 实习学习项目｜Python / FastAPI / Streamlit / Docker**

- 基于 30 天学习计划，独立完成 AI Agent 工程化实践，覆盖 API 封装、RAG、工具调用、部署和前端页面。
- 使用 FastAPI 将 Agent 能力封装为 REST API，支持普通问答、流式响应、健康检查和统一错误处理。
- 使用 Streamlit 构建聊天前端，支持本地模拟模式和后端 API 模式，提升项目可演示性。
- 使用 Dockerfile 和 docker-compose 编写部署配置，并通过健康检查接口验证服务状态。
- 为项目编写详细中文 README、代码注释、练习题和答案，形成可复盘、可展示的 GitHub 作品集。

## 面试讲法

这个项目是我为了系统学习 AI Agent 工程化能力做的一个连续实践项目。
它不是只停留在调用模型，而是把 AI 功能从脚本逐步做成 API、部署配置和前端页面。
我重点练习了 FastAPI 接口化、Streamlit 前端展示、Docker 部署以及本地 fallback 设计。
这个项目让我理解了 AI 应用从“能跑”到“能被别人使用”的完整过程。
